What is the Markowitz model?

The Markowitz model (a.k.a. modern portfolio theory) was pioneered by Nobel Prize-winning American economist Harry Markowitz in 1952.

Background

Considering rising inflation and global market volatility, this project proposes an application that supports private individuals in their investment strategy. Particularly, the application should help its users with creating, managing, and reporting their stock portfolio.

As the hype around stocks like GameStop has underlined the importance of online sentiment around stocks, we plan on not only including hard financial metrics for our application but also Twitter data to provide users with a more comprehensive analysis.

Data Description

The application relies on publicly available financial data from Yahoo Finance, which is obtained via the official Yahoo Finance API. Financial data includes but is not limited to current and historical stock prices, balance sheets, and news feeds. This data can be stored and retrieved as a CSV file.

To support users with a more comprehensive analysis, this model uses publicly available Twitter data obtained via the official Twitter API. This includes but is not limited to tweets with a particular hashtag or tweets from a particular corporate account. This text data can be stored and retrieved as a CSV file.

Lastly, the application also considers user-specific input data, including his/her risk profile, preferred industries, or personally selected stocks. The user-specific data is created as a CSV file or as direct input in the application.

Model Functionalities

This model has five main functionalities:

  1. Creating a Markowitz optimized portfolio for a given list of stocks that the user is interested in. The application comes up with the different weights for each stock and stores the portfolio for the user in a database.
  2. Using the userâ€™s risk profile and preferred industries to create a Markowitz optimized portfolio and store it in a database.
  3. Handling portfolio modifications for stored portfolios, including adding or deleting stocks from the portfolio and recalculating weights accordingly.
  4. Providing comprehensive insights on a given stock, including basic financial ratios and analyses but also more qualitative insights into the current sentiment on Twitter and the news around the stock.
  5. Creating short reports that inform the user about the overall portfolio performance and the current sentiments around the stocks in the portfolio.
